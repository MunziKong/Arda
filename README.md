# ARDA

> 3D TPS Action RPG made with Unity

## 📖 프로젝트 소개

**ARDA**는 Unity 기반으로 개발된 **3D TPS 액션 RPG**입니다.

플레이어는 숲을 수호하는 기계 생명체가 되어 인간 침략자들과 전투를 벌이고, 위험에 처한 몬스터들을 구조하며 다양한 지역을 탐험하게 됩니다.

구조한 몬스터들은 거점으로 이동하여 플레이어의 성장과 탐험을 지원하며, 이를 통해 더욱 강력한 지역에 도전할 수 있습니다.

---

## 🎮 주요 콘텐츠

### ⚔️ 전투 시스템

* 근접 무기(검) 전투
* 원거리 무기(지팡이) 전투
* 기본 공격 콤보 시스템
* 스킬 시스템
* 조준 기반 TPS 전투

### 🐾 구조 시스템

* 구조 장치를 이용한 몬스터 구조
* 구조 성공 시 몬스터 도감 등록
* 다양한 몬스터 수집

### 🌲 탐험 시스템

* 지역 탐험
* 포털을 통한 지역 이동
* 재료 수집
* 적과의 전투

## ⌨️ 조작법

| 키             | 기능            |
| ------------- | ------------- |
| W / A / S / D | 이동            |
| Mouse         | 카메라 회전        |
| Shift         | 달리기           |
| Space         | 점프            |
| C             | 구르기           |
| Left Click    | 기본 공격         |
| Right Click   | 조준            |
| 1 ~ 3         | 스킬 사용         |
| E             | 상호작용          |
| B             | 인벤토리 열기 / 닫기  |
| Q             | 장착 구조 아이템 전환  |
| TAB           | 구조된 몬스터 현황 보드 |
| ESC           | UI 닫기         |

### 테스트용 키

| 키        | 기능                         |
| -------- | -------------------------- |
| F1       | 포션 추가                      |
| F2       | 포션 제거                      |
| F3       | 흔한 구조 키트 획득                |
| Keypad 0 | 게임 초기화                     |
| Keypad 1 | 퀘스트 완료                     |
| Keypad 2 | 10000 Gold 및 모든 아이템 10개 지급 |

## 🗺️ 씬 구성

### Main Scene

플레이어의 거점 역할을 하는 지역입니다.

* 게임 시작
* 포털 이용
* 탐험 준비

### Forest Scene

기본 탐험 지역입니다.

* 적과 전투
* 몬스터 구조

## 🛠️ 구현 기능

### 플레이어
